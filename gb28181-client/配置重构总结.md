# GB28181客户端配置重构总结

## 重构概述

本次重构对 `Gb28181ClientProperties` 配置类进行了全面的优化和重构，主要目标是提升配置的可读性、可维护性和扩展性。

## 主要改进

### 1. 类名优化

- **原类名**: `Gb28181ClientProperties` (冗长且不够简洁)
- **新类名**: `ClientProperties` (简洁明了，符合Spring Boot配置类命名规范)

### 2. 配置前缀优化

- **原前缀**: `sip.gb28181.client` (层级过深)
- **新前缀**: `sip.client` (简洁合理)

### 3. 配置结构重组

#### 3.1 新增配置分组

**设备认证配置 (Auth)**

```yaml
sip:
  client:
    auth:
      device-id: "34020000001320000001"
      device-name: "GB28181-Client"
      username: "admin"
      password: "123456"
      register-expires: 3600
```

**SIP协议配置 (Sip)**

```yaml
sip:
  client:
    sip:
      keep-alive-interval: 60s
      local-port: 5060
      local-address: "127.0.0.1"
      remote-port: 5060
      remote-address: "127.0.0.1"
      transport: "UDP"
      enable-tls: false
```

**网络连接配置 (Network)**

```yaml
sip:
  client:
    network:
      connect-timeout: 10s
      read-timeout: 30s
      write-timeout: 10s
      max-connections: 100
      idle-timeout: 5m
      keep-alive: true
      keep-alive-interval: 30s
```

**重试配置 (Retry)**

```yaml
sip:
  client:
    retry:
      max-retries: 3
      retry-delay: 5s
      retry-multiplier: 2.0
      max-retry-delay: 1m
      enable-exponential-backoff: true
```

#### 3.2 优化现有配置

**性能配置 (Performance)**

- 新增线程池核心线程数和最大线程数分离
- 新增线程池空闲线程存活时间配置
- 增强参数验证

**缓存配置 (Cache)**

- 增加更详细的缓存参数验证
- 优化缓存过期时间配置

### 4. 参数验证增强

#### 4.1 使用Jakarta Validation注解

```java
@NotBlank(message = "设备ID不能为空")
private String deviceId = "34020000001320000001";

@Min(value = 1024, message = "SIP端口不能小于1024")
private int localPort = 5060;

@NotNull(message = "心跳间隔不能为空")
private Duration keepAliveInterval = Duration.ofMinutes(1);
```

#### 4.2 增强验证逻辑

- 端口范围验证 (1024-65535)
- 线程池参数合理性验证
- 缓存大小最小值验证
- 时间参数合理性验证

### 5. 线程池配置优化

**原配置**

```java
private int threadPoolSize = 200;
```

**新配置**

```java
private int coreThreadSize = 10;
private int maxThreadSize = 200;
private Duration threadKeepAliveTime = Duration.ofMinutes(1);
```

### 6. 新增功能特性

#### 6.1 TLS支持

```java
private boolean enableTls = false;
private String tlsCertPath;
private String tlsKeyPath;
```

#### 6.2 指数退避重试

```java
private double retryMultiplier = 2.0;
private Duration maxRetryDelay = Duration.ofMinutes(1);
private boolean enableExponentialBackoff = true;
```

#### 6.3 网络连接池管理

```java

private Duration idleTimeout = Duration.ofMinutes(5);
private boolean keepAlive = true;
```
